# DMC Todo

Hierarchichal todo app.

## Ex

```
[4] School
	[*] Game Sim
	[0] US History
	[*] AP CSP
		[ ] Computering (Due: 32/54/34)
		[ ] Computering alos (Late: 32/54/34)
		[*] Computering alos (Shows for 24h (customizable) after due)
	[2] AP SP
	[0] AP CALC AB
	[0] English III H
	[0] Robotics
	[0] AP Physics 1
```

Non hierarchy mode

```
Late:
	[ ] Computering alos (Late: 32/54/34) (School > AP CSP)

Due:
	[ ] Computering (Due: 32/54/34) (School > AP CSP)

Complete:
	[*] Computering alos (Shows for 24h after due) (School > AP CSP)
```
